# GenAI Chatbot using Gemini (Study Buddy)

This is a beginner-friendly AI chatbot built using **Google Gemini API** and **Streamlit**.

## Features
- Chat UI like WhatsApp
- Answers in simple beginner language
- Gives short examples
- Clear chat button

## Tech Stack
- Python
- Streamlit
- Google Generative AI (Gemini)
- dotenv

1. Install required libraries:
   ```bash
   pip install streamlit google-generativeai python-dotenv
